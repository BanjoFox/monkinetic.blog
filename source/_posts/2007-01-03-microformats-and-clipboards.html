--- title: "microformats and clipboards" layout: post tp_urlid: "6a010534988cd3970b0120a55cea43970b" tp_favoritecount: 0 tp_commentcount: 0 tp_permalink: "http://www.monkinetic.com/2007/01/microformats-and-clipboards.html" --- Michael Kaply [blogs](http://www.kaply.com/weblog/2006/12/live-clipboard-in-operator) about Windows Live Clipboard support for microformats in the Operator plugin for Firefox. This reminds me of the work I did with Brent Simmons [back in the day](http://inessential.com/?comments=1&postid=2032) on an [RSS Item format](http://ranchero.com/netnewswire/rssclipboard.php) for OS X's clipboard, which I supported for a while in my ill-featured-and-ill-fated weblog editor, Sid. I'd like to see something like this come to Safari/Firefox on OS X, so we can easily copy hCal events into iCal, and hCards into AddressBook, etc.
